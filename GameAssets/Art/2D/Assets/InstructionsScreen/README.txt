Instructions page Information
-Appears at the beginning of every playing session, after pressing "play" on the menu screen
-Screens appear sequentially, according to the numbers in their files.  As soon as the user taps to dismiss a screen, the next one pops up.
-When the first instruction screen pops up, a neon x appears in the upper right hand corner.  If tapped, the user skips all the instruction screens.  If not tapped, it remains in the top right corner until the user has dismissed the final screen by tapping anywhere.
-If the player taps the "do not show again", the check appears and the instructions page stops appearing before every game
-Background is semitransparent, and appears over the character selection screen.
